## 4. 핵심 기능
이 서비스의 핵심 기능은 서울시 공공 데이터베이스를 활용한 검색 기능입니다.</br>
이를 이용하여 각자의 취미생활에 맞는 데이터를 검색할 수 있습니다.

### 4.1. Repository

https://github.com/BOWSTRINGHERO/firstCollabor/tree/main

## 5. 핵심 트러블 슈팅
  - csv 데이터가 제대로 import되지 않는 문제
  - DBMS 연결 후 웹 페이지에서 데이터를 불러오지 못하는 문제
  - 데이터베이스를 검색하지 못하는 문제
